# Rorschach

Procedurally generated Rorschach-like image animated in real time using the Processing programming language.

## Launch

When in the root directory, run:

### Linux

```bash
processing-java --sketch=`pwd` --run
```

Or

```bash
processing-java --sketch="$(pwd)" --run
```

### Windows

```powershell
processing-java --sketch="$(pwd)" --run
```
